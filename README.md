# INF.03 - Rozwiązania Egzaminów Zawodowych (2022-2024)

W tym repozytorium znajdują się moje autorskie rozwiązania (niektóre niepełne) praktycznych egzaminów zawodowych **INF.03** (Tworzenie i administrowanie stronami i aplikacjami internetowymi oraz bazami danych) z lat 2022–2024. 

Repozytorium stanowi dokumentację moich przygotowań do końcowego egzaminu w technikum informatycznym.
Jeśli teraz się uczysz to mam nadzieję, że te treści okażą się pomocne.

## Wykorzystane technologie
Zadania egzaminacyjne opierają się na klasycznym stosie technologicznym:
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP
* **Bazy danych:** MySQL

## Uruchomienie
I. Przenieś wybrany folder z zadaniem do katalogu htdocs (w XAMPP) lub www (w WAMP).
II. Uruchom moduły Apache oraz MySQL w panelu kontrolnym.
III. Przejdź pod adres http://localhost/phpmyadmin, utwórz nową bazę danych (zgodną z nazwą w kodzie PHP) i zaimportuj do niej plik .sql dołączony do danego folderu.
IV. Otwórz projekt w przeglądarce, wpisując: http://localhost/nazwa_folderu.
